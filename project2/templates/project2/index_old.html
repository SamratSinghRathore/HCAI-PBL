Welcome to Project 2

This is a Django-based web application designed to demonstrate various features of the Django framework, including file uploads and data visualization.

Project Overview:

  • CSV Upload: Allows users to upload CSV files containing data.
  • Data Processing: Processes the uploaded data to perform calculations or generate plots.
  • Data Visualization: Generates plots like scatter plots based on the uploaded data.

This project aims to show how Django can be used to create a simple but functional web application for data analysis and visualization.
